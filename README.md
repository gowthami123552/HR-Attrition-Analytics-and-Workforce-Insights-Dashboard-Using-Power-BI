# HR-Attrition-Analytics-and-Workforce-Insights-Dashboard-Using-Power-BI
Project Overview
This project focuses on analyzing employee attrition patterns using the IBM HR Analytics dataset. The objective is to identify key factors contributing to employee turnover and provide actionable workforce insights through data analysis and interactive dashboard visualization.The project combines Python-based exploratory data analysis with Power BI dashboard development for business decision support.
Business Problem Statement
Employee attrition impacts organizational productivity, recruitment costs, and workforce stability. Understanding why employees leave helps HR teams design effective retention strategies.
The goal of this project is to:
Calculate overall attrition rate
Identify high-risk departments and job roles
Analyze compensation and overtime impact
Provide HR insights using data visualization
Dataset Information
Dataset: IBM HR Analytics Employee Attrition & Performance
Source: Kaggle (Public Dataset)
Records: 1470 Employees
Features: 35 HR-related attributes
Target Variable: Attrition (Yes / No)
